# How to Download Videos from 123movies for Free (Chrome Browser Extension Method)

If you need a simple way to save videos from supported browser pages, 123movies Downloader gives you a direct workflow: install the extension, open a supported video page, let the extension detect available video files, and choose the quality you want to download.

This guide turns the video walkthrough into a clear step-by-step process for setting up the browser extension and using it correctly.

> Only download videos that you own, have permission to save, or that are legally available for download. Respect copyright, site terms, and local laws.

## What Is 123movies Downloader?

123movies Downloader is a browser extension designed to detect video files on supported pages and make them available through a download button. Instead of copying links manually or searching through browser developer tools, the extension watches for playable video files and lists the available download options inside the browser.

In the walkthrough, the extension is installed manually from a GitHub release package and loaded into Chrome as an unpacked extension.

## Before You Start

You will need:

- Google Chrome or another supported Chromium-based browser.
- Access to the `serpapps/123movies-downloader` GitHub release page.
- The downloaded extension ZIP file for your browser, such as `123movies-downloader-chrome.zip` for Chrome.
- An email address for the extension's one-time verification step.
- A video page where you are authorized to download the content.

## Step 1: Find the 123movies Downloader Release

Open Google and search for:

```text
serpapps 123movies downloader
```

Look for the GitHub repository at:

```text
https://github.com/serpapps/123movies-downloader
```

Open the repository, then find the **Releases** section on the right side of the GitHub page.

The Releases page is where the latest browser extension builds are published.

## Step 2: Download the Chrome Extension ZIP File

On the latest release page, scroll to the downloadable files.

For Chrome, choose the Chrome ZIP asset:

```text
123movies-downloader-chrome.zip
```

Other browser builds follow the same lowercase, hyphenated naming pattern, such as `123movies-downloader-edge.zip` or `123movies-downloader-brave.zip`.

Your browser will download the extension as a ZIP file. Depending on your browser settings, it may save to your desktop, Downloads folder, or another default download location.

## Step 3: Unzip the Extension File

Find the ZIP file you just downloaded and extract it.

On macOS, you can usually double-click the ZIP file. On Windows, right-click it and choose an extraction option such as **Extract All**.

After extraction, keep track of the unzipped folder. You will select that folder when loading the extension into Chrome.

## Step 4: Open the Chrome Extensions Page

Open a new browser tab and go to:

```text
chrome://extensions
```

This page manages installed Chrome extensions, including extensions loaded manually from a local folder.

## Step 5: Turn On Developer Mode

In the top-right corner of the Chrome Extensions page, turn on **Developer mode**.

When Developer mode is enabled, Chrome shows additional extension management buttons, including **Load unpacked**. This is the option you need for installing the unzipped extension folder.

## Step 6: Load the Unpacked Extension

Click **Load unpacked**, then select the extracted extension folder from the ZIP file. For Chrome, that folder comes from `123movies-downloader-chrome.zip`; select the folder, not the ZIP file itself.

Chrome should add the extension to your extensions list. If the extension loads successfully, it is installed and ready to use.

## Step 7: Pin the Extension to the Toolbar

To make the downloader easier to access:

1. Click the puzzle-piece extension icon in the Chrome toolbar.
2. Find the 123movies Downloader extension.
3. Click the pin icon next to it.

The extension icon should now appear directly in your browser toolbar.

## Step 8: Open an Authorized Video Page

Next, open a supported video page where you have the right to save the content.

The extension may detect a video automatically after the page loads. In some cases, the video file will not appear until the media player starts loading the actual stream.

## Step 9: Verify Your Email the First Time You Use the Extension

The first time you open the extension, it may ask you to enter your email address and complete a one-time verification.

The process is:

1. Enter your email address in the extension.
2. Request the verification code.
3. Open your email inbox and copy the code.
4. Paste the code into the extension.
5. Click the verification button to continue.

After verification, the extension should be ready to detect and download available videos.

## Step 10: Play the Video So the Extension Can Detect It

If the extension does not immediately show a downloadable file, click play on the video.

This matters because some video files are only loaded into the browser after playback begins. Once the video starts, the extension can usually detect available sources and show different quality options.

## Step 11: Choose a Video Quality

When the extension detects the video, it may show multiple quality options. Choose the version that fits your needs.

Higher-quality files usually look better but take longer to download and use more storage. Lower-quality files download faster and take up less space.

## Step 12: Click the Download Button

Click the download button for the video option you want.

The extension should show a progress indicator while the file downloads. You can usually continue browsing, open another page, or start a different download while the current one finishes.

When the download is complete, the video will appear in your browser's normal download location.

## Troubleshooting Tips

If the 123movies Downloader extension does not detect a video, try these checks:

- Refresh the page and open the extension again.
- Click play on the video and wait a few seconds.
- Check whether the video is actually loading in the browser.
- Try a different available quality option.
- Confirm that Developer mode is still enabled.
- Make sure you loaded the unzipped extension folder, not the ZIP file itself.
- Reopen `chrome://extensions` and check whether the extension is enabled.

## Where Do Downloaded Videos Go?

Downloaded videos are saved to the same location your browser normally uses for downloads. In Chrome, this is usually your Downloads folder unless you changed the default download location.

To check or change it, open Chrome settings and search for **Downloads**.

## FAQ

### Do I need Developer mode to install the extension?

Yes. Because the walkthrough installs the extension from an unpacked ZIP file rather than directly from the Chrome Web Store, Developer mode must be enabled on the Chrome Extensions page.

### Why do I need to click play before downloading?

Some video pages do not load the actual video file until playback begins. Clicking play gives the browser a chance to request the video source, which allows the extension to detect available download options.

### Can I download multiple videos?

The walkthrough notes that you can continue browsing or start another download while one video is downloading. Download speed and reliability will depend on your browser, network connection, and the size of each file.

### What should I do if the download is slow?

Longer videos and higher-quality files can take several minutes to download. If speed is an issue, try a lower quality option, avoid starting too many downloads at once, and keep the browser open until the download completes.

## Final Thoughts

The 123movies Downloader workflow is straightforward once the extension is installed: download the ZIP from the GitHub release, unzip it, load it through `chrome://extensions`, verify your email, open an authorized video page, press play if needed, and click the download button.

For best results, keep the extension pinned to your toolbar and remember that some pages require playback before downloadable video sources appear.
